# Crypto Lab - Complete Setup

## Overview

A secure environment for managing crypto investments, trading, and experiments.

## Components

### 1. Wallet Management
- `vault.html` - Track wallets by type (cold/hot/newcoin)
- `wallet-manager.js` - CLI wallet management
- Hardware wallet integration ready

### 2. Price Alerts
- Price monitoring for multiple coins
- Telegram/Discord webhook alerts
- SMS alerts (optional)

### 3. Portfolio Tracker
- Multi-chain support
- Real-time price updates
- P&L calculations

### 4. Trading Journal
- Record all trades
- Calculate profits/losses
- Tax prep data

### 5. Gas Tracker
- Monitor gas prices on Ethereum/Base/Solana
- Optimal transaction timing

### 6. New Coin Scanner
- Monitor DEX for new pairs
- Liquidity tracking
- Whale movement alerts

### 7. DeFi Tools
- Staking yields
- Lending rates
- APY comparisons

## Getting Started

1. Set up hardware wallet (Ledger/Trezor)
2. Configure wallet addresses in vault.html
3. Add trading wallet (MetaMask) for hot funds
4. Create isolated wallet for new coins
5. Start tracking!
