## Quick Deploy

### Option 1: pump.fun (Easiest)
1. Go to https://pump.fun
2. Connect wallet (MetaMask/Phantom)
3. Fill in:
   - Name: April
   - Symbol: APRIL
   - Description: AI assistant token
   - Telegram: (optional)
   - Twitter: (optional)
4. Pay ~0.02 SOL (~$1-2)
5. Done! ✅

### Option 2: Remix IDE (Free deployment)
1. Go to https://remix.ethereum.org
2. Create new file: `APRIL.sol`
3. Paste contract code
4. Compile (Ctrl+S)
5. Deploy tab → Injected Provider (MetaMask)
6. Confirm transaction (~0.05 ETH in gas)
7. Done! ✅

### Option 3: Base (Cheaper gas)
1. Go to https://base.org
2. Use the deployer
3. Similar to option 2 but ~$5-10 gas
